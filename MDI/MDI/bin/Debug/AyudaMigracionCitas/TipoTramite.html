AYUDA TIPO TRÁMITE
Insertar: Este botón es parte del objeto navegador, habilita los campos
necesarios para permitir el ingreso de información, además de que
genera automáticamente el código del nuevo registro que se ingresará,
en este formulario es necesario ingresar el tipo de trámite que se
agregará y de seleccionar el estado de este tipo de caso.
Modificar: Este botón es utilizado para poder hacer cambios a
registros dentro de una tabla, ya sea parcial o totalmente, en caso de
que se tenga un registro inhabilitado se modifica por estado activo.
Eliminar: Botón utilizado para eliminar un registro no deseado.
Guardar: Botón utilizado para guardar el contenido ingresado, se
habilita al insertar datos dentro de los campos necesarios, además de
que es importante ingresar todos los campos con información para
poder ejecutar esta operación.
Flechas de navegación: Utilizadas para navegar por los registros
dentro de la tabla.
