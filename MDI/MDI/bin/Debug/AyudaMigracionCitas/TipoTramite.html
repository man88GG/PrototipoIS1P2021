AYUDA TIPO TRMITE
Insertar: Este botn es parte del objeto navegador, habilita los campos
necesarios para permitir el ingreso de informacin, adems de que
genera automticamente el cdigo del nuevo registro que se ingresar,
en este formulario es necesario ingresar el tipo de trmite que se
agregar y de seleccionar el estado de este tipo de caso.
Modificar: Este botn es utilizado para poder hacer cambios a
registros dentro de una tabla, ya sea parcial o totalmente, en caso de
que se tenga un registro inhabilitado se modifica por estado activo.
Eliminar: Botn utilizado para eliminar un registro no deseado.
Guardar: Botn utilizado para guardar el contenido ingresado, se
habilita al insertar datos dentro de los campos necesarios, adems de
que es importante ingresar todos los campos con informacin para
poder ejecutar esta operacin.
Flechas de navegacin: Utilizadas para navegar por los registros
dentro de la tabla.
